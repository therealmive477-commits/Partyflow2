# PartyFlow One-Deploy Version

This is the easiest PartyFlow build.

## What it does
- One GitHub repo
- One Vercel deploy
- No separate backend service
- Create event
- Guest song requests
- Vote songs up and down
- Host page to watch the queue

## Upload to GitHub
Upload every file in this folder to a GitHub repo.

## Deploy to Vercel
- Import the repo into Vercel
- Leave Root Directory blank
- Click Deploy

## Run locally on computer
```bash
npm install
npm run dev
```

Then open:
- http://localhost:3000
- create an event
- use the guest and host links
